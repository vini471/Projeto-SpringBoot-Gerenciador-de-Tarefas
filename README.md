# Projeto Spring Boot - Gerenciador de Tarefas

## 📌 Descrição
Aplicação desenvolvida em **Java com Spring Boot** para gerenciamento de tarefas.  
Permite criar, atualizar e excluir tarefas de forma organizada, facilitando o acompanhamento do progresso.

## 🚀 Funcionalidades
- Criar novas tarefas
- Atualizar informações de tarefas existentes
- Excluir tarefas
- Consultar lista de tarefas

## 🛠️ Tecnologias utilizadas
- Java
- Spring Boot
- Maven
- REST API
